  - `FLOW_SPEED = 0.5` (px per frame at 60fps)
  - `SCROLL_DELTA = 80` (px per wheel event)
- 初始化全局变量：photoElements, rafId, isPaused, manualScrollDelta

### Step 2: 实现照片元素创建 ✅
- 实现 `createPhotoElements()` 函数
- 动态创建 12 个 `.photo-card` 元素
- 设置 data 属性（photoId, direction）
- 创建 img 元素并设置 lazy loading
- 将元素添加到 photoTrack 容器
- 初始化 photoElements 数组

### Step 3: 实现 Z 轴更新和深度模糊 ✅
- 实现 `updatePhotoPositions(deltaZ)` 函数
- 根据 direction 更新照片 Z 位置
- 实现暂停控制（isPaused）
- 实现手动滚动控制（manualScrollDelta）
- 实现 Z 轴边界循环（Z_MIN ↔ Z_MAX）
- 应用 3D transform（translateZ + scale）
- 应用深度模糊效果：`blur(Math.abs(currentZ) / 100)`
- 根据距离设置 opacity

### Step 4: 实现 RAF 动画循环 ✅
- 实现 `startPhotoLoop()` 函数
- 使用 requestAnimationFrame 创建动画循环
- 实现 `stopPhotoLoop()` 函数
- 使用 cancelAnimationFrame 停止循环

### Step 5: 实现滚轮控制 ✅
- 监听 `gallery:wheel` 自定义事件
- 根据滚轮方向设置 manualScrollDelta
- 向下滚动：+SCROLL_DELTA（照片向前移动）
- 向上滚动：-SCROLL_DELTA（照片向后移动）

### Step 6: 实现悬停暂停 ✅
- 监听 mousemove 事件
- 检测鼠标是否悬停在照片卡片上
- 悬停时：设置 isPaused = true，添加 .is-hovered 类
- 离开时：设置 isPaused = false，移除 .is-hovered 类

### Step 7: 初始化 ✅
- 在 DOMContentLoaded 事件中调用 createPhotoElements()
- 监听 `scene:change` 事件
- 当 scene === 3 时启动照片循环
- 当切换到其他场景时停止照片循环

## 技术细节

### 照片数据结构
```javascript
{
  id: number,           // 照片 ID
  src: string,          // 图片路径
  title: string,        // 照片标题
  description: string,  // 照片描述
  date: string,         // 拍摄日期
  initialZ: number,     // 初始 Z 位置
  direction: 1 | -1     // 流动方向
}
```

### RAF 循环架构
```
startPhotoLoop()
  ↓
animate() {
  updatePhotoPositions(manualScrollDelta)
  requestAnimationFrame(animate)
}
  ↓
stopPhotoLoop()
  cancelAnimationFrame(rafId)
```

### Z 轴循环逻辑
```
if (currentZ < Z_MIN) → currentZ = Z_MAX
if (currentZ > Z_MAX) → currentZ = Z_MIN
```

### 深度模糊计算
```
blurAmount = Math.abs(currentZ) / 100
filter: blur(${blurAmount}px)
```
