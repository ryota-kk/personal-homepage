# Scene3 轮播预览更新说明

## 📋 修改概述

将 scene3 的照片放大预览效果从简单的 lightbox 升级为高级的轮播预览（Carousel）。

## 🎯 实现效果

### 保持不变：
- ✅ Scene3 照片墙布局完全保持原样
- ✅ 双列照片流动效果不变
- ✅ 悬停暂停、点击打开功能不变
- ✅ Scene1、Scene2 和其他页面不受影响

### 新增功能：
- ✨ 点击照片后打开沉浸式轮播预览
- ✨ 卡片式 3D 景深效果
- ✨ 流畅的左右切换动画
- ✨ 当前照片居中高亮显示
- ✨ 前后照片缩小透明展示
- ✨ 计数器显示当前位置（如 3 / 12）
- ✨ 支持键盘导航（← / → / ESC）
- ✨ 支持鼠标/触摸拖拽切换
- ✨ 响应式适配桌面端和移动端

## 🎨 设计特点

### 视觉风格
- **深色沉浸式背景**：95% 黑色 + 模糊效果
- **景深式卡片**：当前照片 100% 缩放，前后照片 85% 缩放
- **高级阴影**：多层阴影营造悬浮感
- **流畅动画**：0.6s cubic-bezier 缓动
- **半透明毛玻璃**：信息卡片背景

### 交互体验
- **点击切换**：左右导航按钮
- **键盘控制**：
  - `←` 上一张
  - `→` 下一张
  - `ESC` 关闭预览
- **拖拽切换**：鼠标/触摸拖动 > 100px 触发切换
- **智能禁用**：首张禁用"上一张"，末张禁用"下一张"
- **背景关闭**：点击遮罩层关闭预览

## 📊 技术实现

### 核心类 `GalleryCarousel`
```javascript
class GalleryCarousel {
  constructor(photos)  // 初始化，传入照片数组
  init()               // 创建 DOM 并绑定事件
  createOverlay()      // 动态创建遮罩层和滑块
  open(photoId)        // 打开预览，显示指定照片
  close()              // 关闭预览
  goToNext()           // 切换到下一张
  goToPrev()           // 切换到上一张
  updateSlides()       // 更新滑块位置和状态
}
```

### 数据流
1. `gallery-3d.js` 管理照片墙和 PHOTOS 数组
2. 用户点击照片 → 触发 `gallery:focus` 自定义事件
3. `gallery-carousel.js` 监听事件 → 调用 `open(photoId)`
4. 根据 photoId 定位到对应照片 → 显示轮播预览
5. 用户切换照片 → `updateSlides()` 更新位置和样式
6. 关闭预览 → 恢复照片墙流动

### 兼容性
- ✅ 复用原有的 `window.gallerySystem.pause()`
- ✅ 复用原有的 `window.gallerySystem.resume()`
- ✅ 复用原有的 `gallery:focus` 事件机制
- ✅ 不修改 PHOTOS 数据结构
- ✅ 不影响其他 scene

## 🐛 已知限制

1. **不支持无限循环**：到达第一张/最后一张时不能继续切换（可按需添加）
2. **拖拽阈值固定**：100px 触发切换（可配置化）
3. **照片预加载**：当前未实现预加载相邻照片（可优化性能）
