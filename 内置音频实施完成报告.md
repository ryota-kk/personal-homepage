### 音频文件要求

| 项目 | 建议值 |
|------|--------|
| 格式 | MP3（最佳兼容性） |

| 音量 | 归一化到 -3dB 至 -6dB |


2. 将你的音频文件复制进去
3. 重命名为：`background-music.mp3`

## 🐛 故障排查

### 问题 1：点击进入后没有声音

**检查步骤**：
1. 确认音频文件存在：`ls assets/background-music.mp3`
2. 打开浏览器控制台（F12），查看错误信息
3. 检查是否显示：`✓ 音频自动播放启动`
4. 检查右下角音量滑块是否在 0 位置

**可能原因**：
- ❌ 音频文件不存在或路径错误
- ❌ 浏览器阻止自动播放（应该不会，因为有用户交互）
- ❌ 音频文件格式不支持
- ❌ 音量为 0

**解决方法**：
```javascript
// 在浏览器控制台手动测试
window.audioControllerInstance.play();
```

---

### 问题 2：频谱不跳动

**检查步骤**：
1. 确认能听到声音
2. 查看页面中央是否有频谱柱（白色竖条）
3. 检查控制台是否有错误

**可能原因**：
- ❌ 音频未真正播放
- ❌ Web Audio API 连接问题
- ❌ 频谱位置不在可视区域

**解决方法**：
```javascript
// 检查频谱实例

```

### 问题 3：控制台显示 404 错误

**错误信息**：
```
GET http://localhost:5173/assets/background-music.mp3 404 (Not Found)
```

**原因**：音频文件不存在

**解决方法**：
